# Hostel ERP Management System

A desktop-based Hostel Management System built with C++ and Qt Framework.

## Features
- Secure login system with forgot password option
- Add, view, search, and delete student records
- Room management with availability tracking
- Fee management and tracking
- Complaint registration and viewing

## Tech Stack
- C++ (Qt Framework)
- SQL Database
- Qt Creator IDE

## How to Run
1. Open project in Qt Creator
2. Build and run the project
3. Login with admin credentials
